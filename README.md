📊 Data Analysis Pipeline – Web Scraped Dataset

This project demonstrates a complete data analysis workflow, starting from raw web data extraction to a clean, structured dataset ready for analysis.



🚀 Project Overview

In this project, I worked on collecting and preprocessing data from a website and transforming it into a usable format. The focus is on data cleaning, transformation, and validation, which are critical steps in any data analysis pipeline.



🔧 Steps Performed

1. Data Scraping

Extracted raw data from a website.



2. Data Loading

Loaded and combined data from 3 different pages into a single dataset.



3. Data Cleaning

Removed unnecessary columns to improve dataset quality.



4. Header Management

Appended proper column headers for better readability.



5. Data Type Conversion

Converted columns into appropriate data types for accurate processing.



6. Handling Missing Values

Replaced “-” values with 0 wherever required.



7. Error Checking & Validation

Performed checks to ensure data consistency before applying transformations.



🛠️ Tech Stack

PowerBI

Web Scraping 




📈 Output

Cleaned and structured dataset ready for analysis

Improved data quality and consistency

Reduced errors in further processing
